# Queus | Filas
Aqui temos a implementação de uma fila, do zero, utilizando ponteiros. 
Nessa implementação, utilizo a ideia de de Lista encadeada funcionando como Fila. 

Exemplo:

(Inicio->)| 5 | <- | 4 | <- | 3 | <- | 2 | <- | 1 | ¬ (<- fim)
